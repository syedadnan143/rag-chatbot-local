
# Local AI Knowledge Assistant using RAG (LangChain + FAISS + Local Embeddings)



**A local Retrieval-Augmented chatbot using:**

**- Streamlit**

**- LangChain**

**- Sentence-Transformers (local embeddings)**

**- FAISS (vector search)**



**## Features**

**- Upload PDF/TXT/MD**

**- Chunking + embeddings**

**- Similarity search over your docs**
